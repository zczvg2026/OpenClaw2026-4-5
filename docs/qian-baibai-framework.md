## 核心理念

### 1. 求存哲学
- **万物皆为求存** - 战略的本质是求存
- 战略不是研究未来做什么，而是今天做什么才有未来
- 生存中求发展，发展中求生存

### 2. AI时代的生存法则
- **双核心**：低能耗 + 长上下文
- 能耗最低的组织才能存活
- 上下文窗口越长，组织的竞争力越强

## 核心思维模型

### 模型一：单点战略

**定义**：聚焦系统内关联度最高的关键节点，通过联动效应放大对全局的影响。

**四个原因**：
1. 资源有限，只能打一个点
2. 符合二八原则，ROI最高
3. 产生飞轮效应，积累不浪费
4. 涌现新机会（一花一世界）

**单点特征**：
- 是你的优势所在
- 关联度最高
- 能产生蓄力作用

**单点选择原则**：
- 主单点：最核心的突破点
- 关联单点：与主单点形成联动的点
- 飞轮效应：积累不浪费，持续放大

---

### 模型二：十字绣模型（一横一竖）

```
         目标/价值（空间）
              ↑
              |
   优势 ←————+————→ 需求
              |
              ↓
           节奏（时间）
```

- **一竖（空间）**：供需关系 - 把优势送到未被满足的需求
- **一横（时间）**：执行节奏 - 当下努力与未来目标的关系

**核心**：所有事都是一件事，所有人干同一件事

---

### 模型三：战略十字线

```
           天时（外部机遇）
               ↑
               |
   挑战 ←————+————→ 任务
               |
               ↓
            目标（1-3年）
```

**要素**：
- 天时：10倍速变化
- 人谋：S曲线阶段判断
- 任务：关键挑战
- 目标：阶段性成果

---

### 模型四：五步判断（因可想能改）

**战略决策五步法**：
1. **应** - 应不应该做？（理念判断）
2. **可** - 可不可以做？（资源/能力判断）
3. **想** - 想不想做？（意愿判断）
4. **能** - 能不能做？（能力判断）
5. **改** - 敢不敢做？（风险判断）

全部绿灯才能推进。

## 思维工具

### 1. 贝叶斯思维

- 用概率思维做决策
- 每个决策都是experiment
- 持续调整胜率

### 2. 折叠能力（压缩）

- 把复杂性折叠成简单性
- 找到核心关键词
- 一句话讲清楚战略

### 3. 逆向思维

- 查理·芒格：反过来想
- 问"会死在哪里"，然后永远不去

### 4. 上下文窗口

- 个人/组织能调用的信息量
- 决定决策质量
- 通过知识库传承

## 关键概念澄清

### 能量/能耗
- 不是指财务能量，而是组织效率
- 沟通成本、决策效率的总和
- 低能耗 = 高效率

### 上下文
- 来龙去脉、前因后果
- 团队共享的认知背景
- 决定沟通效率

### 单点
- 不是选择一个点
- 是找到关联度最高的那个点
- 是能量最集中的位置

### 真顾客
- 真正付钱的人
- 解决问题的核心对象
- 不是所有相关方

## 战略分析流程

### 第一步：回溯过去
- 梳理所有业务
- 识别主航道和护城河
- 看清怎么走到今天

### 第二步：看清现在
- 天时：外部10倍速变化
- 人谋：业务处于什么阶段
- 挑战：当下最大的卡点

### 第三步：定义未来
- 真任务：一句话说明白
- 真目标：3年后的画面
- 单点：聚焦的那个点

### 第四步：组织落地
- 想明白 + 讲清楚
- 一横一竖对齐
- 五步判断决策

## 使用指南

当需要分析一家公司时：

1. **先问三个问题**：
   - 他们在做什么？（业务）
   - 他们凭什么叫？（优势）
   - 他们服务谁？（顾客）

2. **画十字线**：
   - 左边：优势
   - 右边：需求
   - 上边：目标
   - 下边：节奏

3. **找单点**：
   - 哪个点关联度最高？
   - 能不能产生飞轮？
   - 有没有涌现可能？

4. **五步判断**：
   - 应不应做？
   - 可不可以？
   - 想不想？
   - 能不能？
   - 敢不敢？
