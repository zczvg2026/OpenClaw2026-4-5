# ⚡ 快速开始

30 秒上手微信文章解析器。

## 1️⃣ 安装依赖

```bash
pip3 install requests beautifulsoup4
```

## 2️⃣ 解析文章

```bash
python3 scripts/wechat_parser.py "https://mp.weixin.qq.com/s/xxxxx"
```

**输出示例：**

```
================================================================================
📰 标题: AI 时代的一人公司
✍️  作者: 某某公众号
🕐 发布时间: 2026-03-10
📊 字数: 3500
🖼️  图片数: 5
================================================================================

📝 正文内容:

这是文章的正文内容...
================================================================================
```

## 3️⃣ 保存到文件

```bash
python3 scripts/wechat_parser.py "URL" --save
```

会生成 `文章标题.json` 文件。
