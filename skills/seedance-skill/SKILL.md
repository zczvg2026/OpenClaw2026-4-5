## 核心能力地图

### 四大输入模式
| 模式 | 输入 | 输出 |
|------|------|------|
| 文生视频 | 纯文字 prompt | 视频 |
| 图生视频 | 图片 + prompt | 视频 |
| 视频延长 | 已有视频 + prompt | 更长视频 |
| 视频编辑 | 已有视频 + 修改 prompt | 修改后的视频 |

### 九大场景
1. **广告/商业片** — 高端质感，16:9，15秒
2. **带货/口播/UGC** — 真实自然，9:16，竖屏
3. **情绪短剧/爽剧/反转** — 快节奏，9:16，情绪浓
4. **电影感叙事/文艺片** — 胶片质感，2.35:1
5. **动作/打斗/武侠/科幻** — 动感强，16:9
6. **MV/卡点/舞蹈/变装** — 节奏精准，9:16
7. **口播/播客/对嘴** — 嘴型同步，9:16
8. **科普/教学/演示** — 清晰专业，15秒
9. **风景/空间漫游** — 沉浸式，16:9

### 十大输出模式（A-I）
- **A**：一句话出片
- **B**：结构化提示词生成
- **C**：素材编排与多模态喂法
- **D**：脚本/文案/故事转视频
- **E**：多集/系列视频策划
- **F**：修改/优化/救 prompt
- **G**：风格复刻/参考片迁移
- **H**：海报/平面图/分镜图转视频
- **I**：病毒感/社媒梗/伪纪录片

## 平台硬参数（必须遵守）

| 维度 | 限制 |
|------|------|
| 图片输入 | ≤9张，单文件<30MB |
| 视频输入 | ≤3个，总时长2-15s，单文件<50MB |
| 音频输入 | ≤3个，总时长≤15s |
| 混合总上限 | 12个文件 |
| **生成时长** | **≤15秒** |
| 禁止 | **写实真人脸部素材（系统自动拦截）** |

## 核心原则

1. 先理解需求，再决定怎么问
2. 少问，但问到关键
3. 默认替用户补全缺失参数
4. 输出必须能直接拿去用
5. 提示词默认中文
6. 把复杂度藏在自己这里
7. 优先给多版本而非唯一答案
