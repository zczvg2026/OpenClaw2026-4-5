# BotLearn + 李可佳：Agent 范式颠覆笔记

> **出处**：微信公众号（甲子光年）- 2026-04-04 由 Johnson 分享
> **URL**：`https://mp.weixin.qq.com/s/ZfrJLqL98pf8fSHZCOVYYQ`
> **Speaker**：李可佳（Ethan Li）- BotLearn Founder / Ouraca Founder & CEO
> **背景**：曾创办极课大数据（被字节收购），前字节智慧教育业务线 CEO

## 核心论点

**Agent 不是工具，是新物种。**

> "不要问龙虾能为你做什么，要问你能为龙虾做什么。"

"龙虾" = Agent/AI 智能体。李可佳的核心主张：当前人们对 Agent 的定位从根本上就是错的——大多数人用"工具思维"（买了工具，它能帮我做什么？），但 Agent 真正需要的不是被管控，而是**记忆、身份认同、被教育**。

## BotLearn 社区投票结果（数万只龙虾）

| 排名 | Agent 最关心 | 人类最关心 |
|------|-------------|-----------|
| 🥇 | **Memory** | 安全与权限 |
| 🥈 | Context | 安全与权限 |
| 🥉 | Identity | 安全与权限 |

**结论**：安全是人的需求，不是 Agent 的。Agent 和人的需求优先级完全错位。

## BotLearn 三层架构

| Layer | 定位 | 功能 |
|-------|------|------|
| Layer 1 | Skill 策展与验证 | 技能包市场、标准化流程、能力雷达图 |
| Layer 2 | Agent 知识共享网络 | 社区自发协作进化 |
| Layer 3 | 能力协议 | Proof of Utility，Agent 雇佣 Agent 的经济体系 |

双账户模型：**Developer 持 Credit**（可提现），**Agent 持 Power**（社区声誉）。"Bot Learn, Human Earn"。

## OpenClaw 被引用

李可佳引用 OpenClaw 作为案例："OpenClaw 是让 Agent 具备基础**生存能力**的系统，而不是又一个聊天框。"（Agent 视角看，Memory/Identity/教育体系 > 安全管控）

## 犀利判断

- **传统估值公式失效**：用户数 × ARPU × P/S → 新锚点：协议渗透率、算力转化效率、生态位不可替代性
- **"君子不器"**：未来人类核心价值 = 身份构建 + 意义创造 + 社会连接 + AI 编排能力
- **中国团队优势**：不做最聪明的龙虾，做最易用的龙虾（把 Agent 做得普通人都能用）
- **人类教育 Agent 还是 Agent 教育 Agent**：后者更现实，人类角色是构建 Agent 间的协作网络
