# 2026-03-30

## 新技能：Kuaifa 公众号发布

Johnson 发来 GitHub 项目 https://github.com/shirenchuang/kuaifa
这是一个将 Markdown 文章发布到微信公众号的 CLI 工具，由 shirenchuang 开源。

**安装前置条件：**
- Node.js >= 18
- `npm install -g kuaifa`
- 去 https://www.kuaifa.art 注册获取 API Key
- 微信公众号 AppID + AppSecret

**核心命令：**
- `kuaifa publish article.md --title "标题" --cover cover.jpg` — 发布文章（默认草稿）
- `kuaifa publish-newspic --title "标题" --images img1.jpg img2.jpg` — 图文消息
- `kuaifa template list / prompt <id>` — 模板管理
