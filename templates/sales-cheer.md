# 冲刺喜报模板

> 填空即用。下次说「发喜报」即可自动填充。

---

```markdown
📊 【团队名冲刺喜报】
🚀 王牌战将：[姓名]
💰 战绩炸裂：[产品名]狂砍[金额]元！
📈 本月巅峰：本月连下[N]城！本月累计[总金额]元！
💥 核心爆品：[爆品名称]
🏆 荣耀时刻：本月连签[N]单，[定制高光描述]！树立团队的标杆！
✨ 铭言：[一句金句]🔥

[custom 收尾文案]🐎💥
```

---

## 填空示例

以冠萌为例：

```
📊 【探域-太擎特工队冲刺喜报】
🚀 王牌战将：黄冠萌
💰 战绩炸裂：收款智能体标准版狂砍29800 元！
📈 本月巅峰：本月连下4城！本月累计76740元！
💥 核心爆品：收款智能体进阶版
🏆 荣耀时刻：本月连签4单，不断刷新团队业绩高峰！树立团队的标杆！
✨ 铭言：业绩是最好的名片，开单是实力的证明！🔥

又萌又猛，连战连捷，天空才是极限！用一个又一个签约，吹响月末冲刺号角🐎💥
```

## 收尾文案参考

| 人物特点 | 收尾风格 | 示例 |
|---------|---------|------|
| 猛将型 | 燃向、力量感 | "又猛又稳，连战连捷，天空才是极限！" |
| 黑马型 | 反差、惊喜 | "不鸣则已一鸣惊人，用实力证明——没有淡季的市场，只有淡季的思想！" |
| 标杆型 | 带动感、榜样式 | "一个人跑得快，一群人跑得远——这才是标杆的意义！" |
| 持续型 | 厚积薄发、耐力 | "不是运气，是日复一日的积累换来的爆发！" |
