# OPC积分商城小程序项目

## 一、产品定位
积分商城小程序，只做积分交易（1元=1积分），不收现金。

## 二、可交易商品
- 实物商品
- 虚拟服务：课程、会员身份、书籍、Token包、AI智能体手机、各类智能体

## 三、会员体系（4级）
| 等级 | 升级条件 | 权益 |
|------|---------|------|
| OPC | 注册即得 | 基础采购价 |
| OPC工会 | 发展5个OPC | 更低价采购 |
| OPC社区 | 发展2个OPC工会 | 更低价采购 |
| OPC城市 | 一年内完成300万业绩（审核制） | 最优惠采购价 |

## 四、分销网络（4级佣金）
- 用户扫码注册 → 终身绑定上下级关系
- A推荐B → B推荐C → C推荐D → D推荐E，A/B/C/D均有佣金
- 佣金来源：等级差价格利润

## 五、交易规则
- 充值积分购物，1:1现金充值
- 各商品单独设计积分价格
- 各等级对应不同采购积分价

## 六、开发方式
AI智能助手（多Agent协作）
目标：1个月内上线

## 七、技术选型
- 后端基础：MedusaJS 生态（mercurjs/mercur 多商户平台）
- 前端：微信小程序
- 分销模块：需自研叠加
